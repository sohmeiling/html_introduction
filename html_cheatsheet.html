Lesson 02

HTML Basics

Learn how to create the structure of a web page using HyperText Markup Language — the foundation of every website on the Internet.

01

What is HTML?

HTML (HyperText Markup Language) is the language used to create web pages. It tells the browser what content to show and how to organise it. HTML uses tags written in angle brackets. Most tags come in pairs: an opening tag <p> and a closing tag </p>.

HTML
<p>If <b>Tetris</b> taught me anything, it's that
<i>mistakes accumulate</i> and <i>achievements disappear</i>.</p>
Result in browser
If Tetris taught me anything, it's that mistakes accumulate and achievements disappear.
02

Document structure

Every HTML file must follow a standard skeleton so the browser can read it correctly.

HTML
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello world!</h1>
    <p>This is my first web page.</p>
  </body>
</html>
TagPurpose
<!DOCTYPE html>Declares the document as HTML5. Always the first line.
<html>Root container for the entire page.
<head>Holds page metadata — not shown in the browser window.
<meta charset="UTF-8">Ensures special characters display correctly.
<title>Text shown on the browser tab.
<body>Everything visible in the browser window.
03

Headings

HTML provides six heading levels, from <h1> (largest) to <h6> (smallest). Each is a block element — it starts on a new line.

HTML
<h1>Heading 1 — Page title</h1>
<h2>Heading 2 — Main section</h2>
<h3>Heading 3 — Sub-section</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
Best practice: Use only one <h1> per page. Use <h2>–<h6> in order for sub-sections.
04

Paragraphs

The <p> tag defines a paragraph. The browser automatically adds spacing above and below each one.

HTML
<p>The Internet is a global computer network.</p>
<p>Web developers work in two areas: frontend and backend.</p>
05

Text formatting

These inline elements style parts of your text without starting a new line.

Bold

HTML
<p><b>Danger:</b> falling rocks.</p>
<!-- <strong> = bold AND important (preferred) -->
<p><strong>Danger:</strong> falling rocks.</p>

Italic

HTML
<p>I <i>love</i> programming!</p>
<!-- <em> = italic AND emphasised (preferred) -->
<p>I <em>love</em> programming!</p>
Tip: Both <b>/<i> and <strong>/<em> look the same in the browser. Prefer <strong> and <em> — they also signal meaning to screen readers and search engines.
06

Blockquote

Use <blockquote> to display a long quotation. The browser indents it by default.

HTML
<p>A quote from <b>Steve Jobs</b>:</p>
<blockquote>
  Computers are the most amazing tools I've ever
  encountered. They're the bicycle for our minds.
</blockquote>
07

Links

The <a> (anchor) tag creates a clickable link. The href attribute specifies where it goes.

HTML
<!-- Basic link -->
<a href="https://en.wikipedia.org/wiki/HTML">What is HTML?</a>

<!-- Open in a new tab -->
<a href="https://wikipedia.org" target="_blank">Wikipedia</a>

<!-- Link to another page in the same site -->
<a href="page3.html">Go to CSS lesson</a>
08

Images

The <img> tag embeds an image. It is self-closing — no closing tag needed.

HTML
<img src="cat.jpg" alt="A fluffy orange cat">

<!-- From the internet -->
<img src="https://placehold.co/300x200"
     alt="Placeholder"
     width="300" height="200">
Always write alt. It is read by screen readers for visually impaired users, and it appears when the image fails to load.
09

Unordered list

A <ul> displays items with bullet points. Use it when order does not matter. Each item uses <li>.

HTML
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
10

Ordered list

A <ol> displays items with sequential numbers. Use it when order matters, such as steps.

HTML
<ol>
  <li>Write the HTML</li>
  <li>Add CSS styling</li>
  <li>Publish to the web</li>
</ol>
11

Nested lists

Place a list inside a <li> to create a sub-list. You can mix <ul> and <ol>.

HTML
<ol>
  <li>Walk the dog</li>
  <li>Go to the store and buy:
    <ul>
      <li>Notebooks</li>
      <li>A stapler</li>
    </ul>
  </li>
  <li>Have dinner</li>
</ol>
12

Self-closing tags

Some elements have no content and no closing tag.

HTML
<p>Line one.<br>Line two.</p>  <!-- line break -->
<hr>                          <!-- horizontal line -->
<img src="photo.jpg" alt="..."> <!-- image -->
13

Containers not in slides

<div> and <span> are the most commonly used HTML elements. They are invisible on their own — used to group content so CSS can style it.

HTML
<!-- div: block container (starts on new line) -->
<div>
  <h2>About me</h2>
  <p>I am a web developer.</p>
</div>

<!-- span: inline container (stays in the line) -->
<p>My favourite colour is <span>blue</span>.</p>
14

Comments

Comments are notes in your code. The browser ignores them completely.

HTML
<!-- This is a comment. Not shown in the browser. -->
<p>Hello world!</p>

<!-- Disable code temporarily:
<p>This paragraph is hidden.</p>
-->
15

Tag closing rules

Nested tags must close in the reverse order they were opened — last in, first out.

HTML
<!-- WRONG -->
<p><i>Be careful!</p></i>

<!-- CORRECT -->
<p><i>Be careful!</i></p>
Common mistake: Forgetting to close a tag can break the layout of the entire page.
