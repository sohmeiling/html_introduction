Introduction to HTML

Learn how to create the structure of a web page using HyperText Markup Language — the foundation of every website on the internet.

01

What is HTML?

HTML (HyperText Markup Language) is the language used to create web pages. It tells the browser what content to show and how to organise it — headings, paragraphs, images, links, and more.

HTML uses tags written in angle brackets, like <p> for a paragraph. Most tags come in pairs: an opening tag <p> and a closing tag </p>. Everything between them is the content.

HTML 
<p>If <b>Tetris</b> taught me anything, it's that
<i>mistakes accumulate</i> and <i>achievements disappear</i>.</p>
Result in browser
If Tetris taught me anything, it's that mistakes accumulate and achievements disappear.
02

Document structure

Every HTML file must follow a standard structure so the browser can read it correctly. Here is the skeleton of a complete HTML page:

HTML 
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello world!</h1>
    <p>This is my first web page.</p>
  </body>
</html>

What each part does

TagPurpose
<!DOCTYPE html>Tells the browser this is an HTML5 document. Always the first line.
<html>The root container for the entire page.
<head>Holds information about the page (not displayed in the browser window).
<meta charset="UTF-8">Ensures special characters (accents, emoji, Asian scripts) display correctly.
<title>The text shown on the browser tab.
<body>Everything visible in the browser window goes here.
Tip: Always include <meta charset="UTF-8"> in your <head>. Without it, characters like é, ñ, 你好, or emoji may appear broken.
03

Headings

HTML provides six levels of headings, from <h1> (largest, most important) to <h6> (smallest). Headings are block elements — each one starts on a new line.

HTML 
<h1>Heading 1 — Page title</h1>
<h2>Heading 2 — Main section</h2>
<h3>Heading 3 — Sub-section</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
Result in browser

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5 & Heading 6 (smaller)

Best practice: Use only one <h1> per page (the main title). Use <h2>–<h6> in order for sub-sections.
04

Paragraphs

The <p> tag defines a block of paragraph text. The browser automatically adds a blank space above and below each paragraph.

HTML 
<p>The Internet is a global computer network designed
for storing, processing, and transmitting information.</p>

<p>Web developers work in two areas: frontend and backend.</p>
Result in browser

The Internet is a global computer network designed for storing, processing, and transmitting information.

Web developers work in two areas: frontend and backend.

05

Text formatting

These inline elements let you style parts of your text without starting a new line.

Bold text

HTML 
<p><b>Danger:</b> falling rocks.</p>

<!-- <strong> means bold AND important (preferred) -->
<p><strong>Danger:</strong> falling rocks.</p>
Result in browser

Danger: falling rocks.

Italic text

HTML 
<p>I <i>love</i> programming!</p>

<!-- <em> means italic AND emphasised (preferred) -->
<p>I <em>love</em> programming!</p>
Result in browser

I love programming!

Tip — <b> vs <strong>: Both look the same in the browser, but <strong> also signals importance to screen readers and search engines. Prefer <strong> and <em> in modern HTML.
06

Blockquote

Use <blockquote> to display a long quotation from another source. The browser indents it by default.

HTML 
<p>A quote from <b>Steve Jobs</b>:</p>
<blockquote>
  Computers are the most amazing tools I've ever
  encountered. They're the equivalent of a bicycle
  for our minds.
</blockquote>
Result in browser

A quote from Steve Jobs:

Computers are the most amazing tools I've ever encountered. They're the equivalent of a bicycle for our minds.
07

Links

The <a> (anchor) tag creates a clickable hyperlink. The href attribute specifies where the link goes.

HTML 
<!-- Basic link -->
<a href="https://en.wikipedia.org/wiki/HTML">
  What is HTML?
</a>

<!-- Open in a new browser tab -->
<a href="https://en.wikipedia.org"
   target="_blank">Wikipedia</a>

<!-- Link to another page in the same folder -->
<a href="about.html">About me</a>

Attributes

AttributeDescription
href="..."The URL the link points to. Required.
target="_blank"Opens the link in a new browser tab.
08

Images

The <img> tag embeds an image. It is a self-closing tag — it has no closing tag. The src attribute specifies the image file path or URL.

HTML 
<!-- Image from the same folder -->
<img src="cat.jpg" alt="A fluffy orange cat">

<!-- Image from the internet -->
<img src="https://placehold.co/300x200"
     alt="Placeholder image">

<!-- Set width and height -->
<img src="logo.png"
     alt="Company logo"
     width="200"
     height="100">

Attributes

AttributeDescription
src="..."Path or URL of the image. Required.
alt="..."Text description of the image. Required for accessibility.
width="..."Image width in pixels. Optional.
height="..."Image height in pixels. Optional.
Tip: Always write an alt attribute. It is read aloud by screen readers for visually impaired users, and it appears when the image fails to load.
09

Unordered list

An unordered list (<ul>) displays items with bullet points. Use it when the order of items does not matter. Each item is wrapped in a <li> tag.

HTML 
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
Result in browser
  • HTML
  • CSS
  • JavaScript
10

Ordered list

An ordered list (<ol>) displays items with sequential numbers. Use it when the order matters, such as steps in a process.

HTML 
<ol>
  <li>Write the HTML</li>
  <li>Add CSS styling</li>
  <li>Publish to the web</li>
</ol>
Result in browser
  1. Write the HTML
  2. Add CSS styling
  3. Publish to the web
11

Nested lists

You can place a list inside a <li> element to create a sub-list. You can mix <ul> and <ol> freely.

HTML 
<ol>
  <li>Walk the dog</li>
  <li>Go to the store and buy:
    <ul>
      <li>Notebooks</li>
      <li>A stapler</li>
    </ul>
  </li>
  <li>Have dinner</li>
</ol>
Result in browser
  1. Walk the dog
  2. Go to the store and buy:
    • Notebooks
    • A stapler
  3. Have dinner
12

Self-closing tags

Some HTML elements do not have content, so they do not need a closing tag. These are called self-closing (or void) elements.

HTML 
<!-- Line break: moves text to the next line -->
<p>Line one.<br>Line two.</p>

<!-- Horizontal rule: draws a divider line -->
<hr>

<!-- Image: no closing tag needed -->
<img src="photo.jpg" alt="A photo">
TagPurpose
<br>Line break inside a paragraph
<hr>Horizontal divider line
<img>Embed an image
<meta>Metadata in the <head>
13

Containers not in slides

<div> and <span> are the most commonly used HTML elements. On their own they do nothing visible — they are used to group content so you can style it with CSS later.

<div> — block container

Groups block-level elements. Starts on a new line and takes up the full width.

HTML 
<div>
  <h2>About me</h2>
  <p>I am a web developer.</p>
</div>

<span> — inline container

Groups inline content (e.g. one word inside a sentence). Stays on the same line.

HTML 
<p>My favourite colour is <span>blue</span>.</p>
Tip: <div> and <span> are building blocks for layout. You will use them constantly once you start writing CSS.
14

Comments

Comments are notes you write in your code for yourself (or your team). The browser ignores them completely — they are never displayed on the page.

HTML 
<!-- This is a comment. The browser ignores it. -->
<p>Hello world!</p>

<!-- TODO: add an image here later -->

<!-- You can also comment out code to disable it:
<p>This paragraph is hidden.</p>
-->
15

Tag closing rules

Nested tags must be closed in the reverse order they were opened — last in, first out.

HTML 
<!-- WRONG: <i> was opened after <p>, so it must close first -->
<p><i>Be careful!</p></i>

<!-- CORRECT -->
<p><i>Be careful!</i></p>
Common mistake: Forgetting to close a tag can break the layout of your entire page. Always double-check that every opening tag has a matching closing tag in the right order.
