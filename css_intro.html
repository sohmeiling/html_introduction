Lesson 03

CSS Basics

Make your HTML pages look great with Cascading Style Sheets — control colours, fonts, spacing, and layout.

01

What is CSS?

CSS (Cascading Style Sheets) is the language used to style HTML pages. While HTML defines the structure of a page, CSS controls its appearance — colours, fonts, spacing, and layout.

A CSS rule has two parts: a selector (which element to style) and one or more declarations (what to change).

CSS syntax
selector {
  property: value;
  property: value;
}

/* Example */
p {
  color: blue;
  font-size: 16px;
}
02

Adding CSS to HTML

There are three ways to add CSS to an HTML page.

Method 1 — External stylesheet (recommended)

Write CSS in a separate .css file, then link it in the <head>. Best for real projects — keeps HTML and CSS separate.

HTML — index.html
<head>
  <link rel="stylesheet" href="style.css">
</head>

Method 2 — Internal stylesheet

Write CSS inside a <style> tag in the <head>. Useful for small or single-page projects.

HTML
<head>
  <style>
    p {
      color: navy;
    }
  </style>
</head>

Method 3 — Inline style

Apply CSS directly to a single element using the style attribute. Avoid this for large projects — hard to maintain.

HTML
<p style="color: red; font-size: 18px;">Hello!</p>
Best practice: Use an external stylesheet for most projects. It keeps your code organised and lets you style multiple pages from one file.
03

Selectors — element, class, and ID

Selectors tell CSS which HTML elements to style.

Element selector

Selects all elements of a given tag. Affects every matching element on the page.

CSS
p {
  color: grey;         /* all <p> elements */
}

h1 {
  font-size: 2rem;    /* all <h1> elements */
}

Class selector ( . )

Selects elements that have a specific class attribute. The same class can be used on multiple elements. Prefix the class name with a dot.

CSS + HTML
/* CSS */
.highlight {
  background-color: yellow;
  font-weight: bold;
}

/* HTML */
<p>Normal text. <span class="highlight">Highlighted text.</span></p>
<p class="highlight">This whole paragraph is highlighted too.</p>

ID selector ( # )

Selects a single unique element with a specific id. Each ID must be used only once per page. Prefix with a hash symbol.

CSS + HTML
/* CSS */
#main-title {
  color: darkblue;
  text-align: center;
}

/* HTML */
<h1 id="main-title">Welcome to my site</h1>
SelectorSyntaxMatches
Elementp { }All <p> tags
Class.highlight { }All elements with class="highlight"
ID#main-title { }The one element with id="main-title"
04

Combining selectors

You can write rules that apply to multiple selectors at once, or target elements inside other elements.

CSS
/* Apply the same rule to multiple selectors */
h1, h2, h3 {
  font-family: Georgia, serif;
}

/* Target <a> tags that are inside a <nav> */
nav a {
  color: white;
  text-decoration: none;
}

/* Target <p> tags that have class="intro" */
p.intro {
  font-size: 1.1rem;
}
05

Colors

CSS supports several ways to define colours. You can set the text colour with color and the background with background-color.

CSS
h1 {
  color: red;                /* named colour */
  color: #c84b2f;            /* hex code */
  color: rgb(200, 75, 47);  /* RGB values (0–255) */
}

body {
  background-color: #f9f7f4;  /* light cream background */
}

06

Text and fonts

CSS
p {
  font-family: Arial, sans-serif;  /* typeface */
  font-size:   16px;               /* size in pixels */
  font-weight: bold;               /* normal | bold | 100–900 */
  font-style:  italic;             /* normal | italic */
  line-height: 1.6;               /* spacing between lines */
  text-align:  center;            /* left | center | right */
  text-decoration: underline;     /* none | underline */
  letter-spacing: 0.05em;        /* space between letters */
}
Live example

font-family: Georgia — elegant serif font.

font-size: 1.2rem + font-weight: bold + color: accent.

text-align: center · letter-spacing · text-transform: uppercase

Google Fonts: You can use free web fonts by adding a <link> to fonts.googleapis.com in your <head>, then referencing the font name in font-family.
07

The box model

Every HTML element is a rectangular box. The box model describes the four layers that make up that box — from inside to outside:

margin — space outside the border
border — the element's border
padding — space inside the border
content
CSS
div {
  margin:  20px;           /* space outside (all sides) */
  padding: 16px;           /* space inside (all sides) */

  /* Control each side individually */
  margin-top:    10px;
  margin-right:  20px;
  margin-bottom: 10px;
  margin-left:   20px;

  /* Shorthand: top right bottom left */
  margin: 10px 20px 10px 20px;

  /* Shorthand: top/bottom left/right */
  margin: 10px 20px;
}
08

Borders

CSS
p {
  border: 1px solid black;        /* width style colour */
  border-radius: 8px;             /* rounded corners */
  border-left: 4px solid navy;   /* one side only */
}
Examples

border: 1px solid — standard box border with rounded corners.

border-left: 4px solid — accent line on the left only.

09

Width and height

CSS
img {
  width:  300px;    /* fixed width in pixels */
  height: 200px;    /* fixed height in pixels */
}

.container {
  width:     80%;    /* percentage of parent width */
  max-width: 800px;  /* never wider than 800px */
  margin:    0 auto; /* horizontally centred */
}
Responsive tip: Use max-width instead of width for containers. The page will look good on both large screens and small mobile screens.
10

Display property

The display property controls how an element behaves in the flow of the page.

CSS
.box { display: block; }   /* full-width, starts on new line */
.tag  { display: inline; }  /* sits within text flow */
.hide { display: none; }   /* hidden, takes no space */
ValueBehaviourDefault for
blockStarts on a new line, full width<div>, <p>, <h1>
inlineSits inside text, width = content<span>, <a>, <b>
inline-blockInline position, but accepts width/heightButtons, badges
noneCompletely hiddenHidden menus

Backgrounds

CSS

  background-color: #f9f7f4;
}

.hero {
  background-image:    url("banner.jpg");
  background-size:     cover;      /* fill the box */

}
